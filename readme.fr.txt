Split Email Providers est un addon WooCommerce qui envoie des emails automatisés aux fournisseurs pour chaque commande, simplifiant ainsi la communication.

== Description ==

🌟 **Split Email Providers** est un addon WooCommerce innovant qui optimise le suivi des commandes pour les gérants de boutiques ayant plusieurs fournisseurs. Cet addon envoie des emails automatisés pour chaque commande, à chaque fournisseur correspondant. Plus besoin de passer par une marketplace, ce qui rend le processus plus simple, rapide et efficace. Idéal pour les entreprises qui souhaitent garder une relation directe avec leurs fournisseurs tout en utilisant WooCommerce.

Fonctionnalités principales :
* 📬 **Envoi d’emails automatiques aux fournisseurs** : À chaque commande, les emails sont envoyés aux fournisseurs concernés avec les détails de leurs produits.
* 🔄 **Intégration fluide** : Fonctionne directement avec WooCommerce sans nécessiter de configuration complexe.

== Installation ==
 
1. **Téléchargez l’addon** depuis le répertoire WordPress ou directement depuis notre site.  
2. **Accédez à votre tableau de bord WordPress**, cliquez sur « Extensions > Ajouter » et téléversez le fichier.  
3. **Activez l’addon**, attribuez vos produits aux fournisseurs, et envoyez des emails automatisés à chaque fournisseur concerné.  

== Frequently Asked Questions ==

= 🛠️ Est-ce que cet addon est uniquement pour WooCommerce ? =  
Oui, Split Email Providers est conçu spécifiquement pour les boutiques WooCommerce.  

= 📝 Peut-on personnaliser le contenu des emails envoyés aux fournisseurs ? =  
Non, l'addon n'inclut pas pour l'instant d'options pour personnaliser le message et les détails de l'email.
